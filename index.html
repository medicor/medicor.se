Medicor's Playground

Me, playing around with the evolving Web Platform. Hosted on my Radxa Rock. Developed with Cloud9 IDE.

This app is using flexbox layout without media queries (works somewhat). Also using Bliss and Polyfill.io as light sugaring. Always using Simple Material Design Palette.
NARA
A simple two-page, server-side rendered app built with Node.js, Express, Express Handlebars, MongoDB and Bootstrap. Also free-hosted at OpenShift (which means it can take some time to wake up from hybernation ).
Stratum blog
A blog built with Hexo, which can be setup as a static site generator like Jekyll (but Node.js), and as a live server as in this case. Uses my Hexo theme Lightpush, my Hexo filter hexo-filter-api and the great, Ghost like admin tool, hexo-admin.
HSA search
A Polymer app to search the HSA catalog (Swedish healthcare entities). Data is cached in a local MongoDB database, accessed through a mongodb-rest interface, since their api does not provide CORS (as far as I know).
Soundcloud Unfollow
A vanilla Web Platform app, that lets a SoundCloud user list all users that are followed but aren't following back. Just using the SoundCloud API and Polyfill.io for polyfilling legacy browsers. Some CSS transitions and plain JavaScript dribbling.
TallyHo!
Simple time tracking and invoicing with Polymer and Firebase. Hosted on Firebase.
